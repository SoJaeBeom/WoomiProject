## 기능

- 로그인
- 회원 가입
- 게시판 카테고리
  - MBTI 별로 4가지 카테고리 구성
- 전체 게시판 목록 조회
- 게시글 조회 / 작성 / 수정 / 삭제
- 댓글 추가 / 수정 / 삭제
- 검색 기능
